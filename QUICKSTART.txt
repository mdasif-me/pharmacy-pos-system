                        üê≥ ALTERNATIVE: Use Docker
===============================================================================

If you prefer building locally on Ubuntu:

**One-time setup:**
```bash
sudo apt-get update
sudo apt-get install -y docker.io
sudo usermod -aG docker $USER
newgrp docker
```

**Build Windows app:**
```bash
docker build -f Dockerfile.windows -t pharmacy-pos-windows .
docker run --rm -v $(pwd)/release-windows:/project/release-windows pharmacy-pos-windows
```

Your Windows build will be in: `release-windows/win-unpacked/`
